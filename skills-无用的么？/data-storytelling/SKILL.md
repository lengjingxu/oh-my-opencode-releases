# 数据分析可视化 Skill

## 功能
- 数据探索和分析
- 生成可视化图表
- 创建数据报告
- 数据洞察提取

## 使用方式

### 数据分析
```
分析这份销售数据，找出关键趋势
```

### 生成图表
```
用柱状图展示每月销售额
```

### 创建报告
```
生成一份数据分析报告
```

## 支持的图表类型

| 图表类型 | 适用场景 |
|----------|----------|
| 折线图 | 趋势变化 |
| 柱状图 | 分类比较 |
| 饼图 | 占比分布 |
| 散点图 | 相关性分析 |
| 热力图 | 密度分布 |
| 漏斗图 | 转化分析 |

## 数据叙事原则

### 1. 明确目标
- 这个数据要回答什么问题？
- 目标受众是谁？

### 2. 选择正确的可视化
- 比较 → 柱状图
- 趋势 → 折线图
- 占比 → 饼图
- 关系 → 散点图

### 3. 突出关键信息
- 使用颜色强调重点
- 添加注释说明
- 去除无关信息

### 4. 讲述故事
- 开头：背景和问题
- 中间：数据发现
- 结尾：结论和建议

## 输出格式

### HTML 图表
使用 ECharts 或 Chart.js 生成交互式图表

### 静态图片
使用 Python matplotlib 或 Node.js canvas 生成 PNG

### 数据报告
Markdown 格式，包含图表和文字说明
