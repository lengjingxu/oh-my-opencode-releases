# Oh-My-OpenCode 配置中心

一站式安装和配置工具。打开应用即可自动完成所有安装和配置。

## 功能

### 安装向导
1. **自动安装 OpenCode** - 检测并安装 OpenCode 核心程序
2. **自动安装 Oh-My-OpenCode** - 安装多 Agent 协作插件
3. **技术栈选择** - 选择前端/后端技术栈，自动配置提示词
4. **API Key 配置** - 配置模型服务 API Key
5. **个性化设置** - 设置 AI 对你的称呼

### 配置管理
- **提示词配置** - 为不同 Category 配置专属提示词
- **全局规范编辑** - 编辑 AGENTS.md 文件
- **设置管理** - 配置 API Key、打开配置目录

## 使用方式

### 方式一：直接运行（开发模式）

```bash
cd config-app
npm install
npm start
```

### 方式二：打包后运行

```bash
cd config-app
npm install
npm run build:mac   # macOS
npm run build:win   # Windows
npm run build:linux # Linux
```

打包后的应用在 `dist/` 目录下。

## 安装流程

```
打开应用 → 检测安装状态 → 安装 OpenCode → 安装 Oh-My-OpenCode 
    → 选择技术栈 → 配置 API Key → 设置称呼 → 完成！
```

用户只需要：
1. 下载并打开应用
2. 按向导完成配置
3. 开始使用 `opencode` 命令

## 技术栈

- Electron 28
- Vue 3 (CDN)
- Tailwind CSS + DaisyUI
