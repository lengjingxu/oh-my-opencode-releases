多 Agent 协作

不同 Agent 各有所长，协作完成复杂任务。就像一个团队：有人负责探索、有人负责思考、有人负责执行。

Sisyphus

主执行者，负责任务编排和代码实现

擅长：任务分解、代码编写、流程控制
推荐：Claude Opus / Sonnet

Oracle

高智商顾问，处理复杂架构和疑难问题

擅长：架构设计、疑难调试、技术决策
推荐：Claude Opus / o1

Librarian

文档专家，查阅外部资料和最佳实践

擅长：查文档、找示例、搜开源代码
推荐：Gemini Pro / Flash

Explore

代码探索者，快速理解项目结构

擅长：代码搜索、模式发现、上下文理解
推荐：Gemini Flash

协作示例：实现用户认证功能

❌ 传统方式（串行）
1. 探索代码结构 // 5分钟
2. 查 JWT 文档 // 3分钟
3. 查数据库 schema // 2分钟
4. 开始写代码 // 15分钟
总计：25+ 分钟
✅ Enchant 方式（并行）
Explore → 探索代码结构
Librarian → 查 JWT 最佳实践
Sisyphus → 同时开始写代码
↓ 信息汇总后继续
总计：8 分钟
